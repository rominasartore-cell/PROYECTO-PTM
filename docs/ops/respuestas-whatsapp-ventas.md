# PTM - Respuestas WhatsApp y ventas

## Respuesta inicial

Hola. Puedes revisar tus multas en Prescribe tu Multa subiendo tu certificado de multas de transito. El sistema realiza una revision preliminar y, si corresponde, puedes comprar un paquete con informe, instructivo y escritos base para solicitar la prescripcion.

## Si pregunta si se eliminan las multas

No puedo prometer eliminacion de multas. Lo que se entrega es un analisis de posible prescripcion y documentos base para presentar la solicitud ante el organismo correspondiente. La decision final no depende de nosotros.

## Si pregunta cuanto cuesta

El valor del paquete digital aparece en la pagina antes de pagar. Incluye informe, instructivo y escritos base cuando el analisis detecta multas potencialmente prescritas.

## Si pregunta que necesita

Necesitas tener tu certificado de multas de transito no pagadas en PDF y subirlo en la pagina. Luego revisas el resultado preliminar y decides si compras el paquete.

## Si pregunta si sirve para cualquier multa

El sistema esta pensado principalmente para revisar multas de transito informadas en certificado. Si el caso es especial, confuso o no aparece claro en el documento, conviene revisarlo manualmente antes de comprar.

## Si pregunta si incluye abogado en juicio

El paquete digital no implica patrocinio judicial ni representacion en audiencia, salvo que eso se acuerde por separado. Incluye documentos base e instrucciones para orientar la presentacion.

## Si el certificado es ilegible

Si el certificado no se lee bien, el analisis puede fallar o ser incompleto. Lo ideal es descargar un certificado claro y actualizado antes de usar la pagina.

## Si quiere pagar por fuera

Por orden y seguridad, el pago debe hacerse por el flujo de la pagina. Asi queda asociado a tu solicitud y podemos generar correctamente los documentos.

## Si pregunta cuando recibe documentos

Despues del pago debes completar tus datos para documentos. Luego se revisa la informacion y se prepara la entrega. En operacion controlada se revisa manualmente antes de enviar.

## Si llega un caso complejo

Ese caso necesita revision manual antes de confirmar si corresponde usar el flujo digital. Es mejor revisarlo con calma que prometer algo que el documento no permite sostener.
