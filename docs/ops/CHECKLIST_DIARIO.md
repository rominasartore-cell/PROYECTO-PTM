# PTM - Checklist operativo diario

## Revisión diaria rápida

1. Entrar al admin.
2. Revisar pagos aprobados.
3. Revisar pagos pendientes extraños.
4. Revisar correos enviados.
5. Revisar solicitudes con pago aprobado y documentos pendientes.
6. Revisar errores reportados por clientes.
7. Revisar Search Console si hay alertas.
8. Revisar GA4: visitas, análisis iniciados, compras iniciadas y pagos aprobados.

## Estados relevantes

- payment_status approved: cliente pagó.
- purchase_status paid: compra confirmada.
- documents_ready: documentos fueron enviados.
- rejected: pago rechazado.
- pending: preferencia creada pero no pagada.

## Regla

Pago aprobado sin documentos enviados = prioridad alta.
Pago pendiente antiguo = revisar, pero no molestar salvo que haya contacto del cliente.
